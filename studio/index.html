Studio

An independent Unreal Engine studio from Germany

Exovision Games creates medieval worlds, systems, tools and assets in Unreal Engine with a small-studio focus on believable play and practical production.

Who we are

Who we are

Exovision Games / Exovision is a small independent German studio led by Florian Versteegen. The main project is ARKEM, while the broader work focuses on Unreal Engine, medieval worlds and production-ready tools.

The studio favors believable medieval worlds, modular systems, practical production tools and gameplay-first design over oversized promises.

What we build

What we build

A focused set of work areas keeps the studio practical and grounded.

Original games

ARKEM is the flagship project: a medieval single-player strategy RPG currently in development.

Unreal Engine tools

Modular building workflows, gameplay systems and production helpers built for real implementation.

Marketplace assets

Production-minded assets and utilities for developers building playable worlds.

Technical art and gameplay systems

Integration-focused technical art, Blueprint/C++ systems and strategy-RPG mechanics.
